The sampled repositories are listed in file "sampled_repositories.txt",
where each line is a repository, the fields are separated by ";",
the first filed is the group the repository belong to, and the second field is the full name of the repository.

The merged names and emails are listed in file "developers_names_emails.txt",
where each line is a developer, names and emails are separated by "|-:+|",
names are separated by "|-;+|", and emails are separated by |-;+|.
**Note: This file is provided for paper review. Please don't use it for other purposes. When publishing the dataset, we will mask the postfixes of the emails for developers' privacy.**
